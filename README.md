## GMM_estimator
基于 GMM 的语音关键词识别系统开发
项目内容：针对0-9数字、O共11类语音指令，独立完成传统语音关键词识别系统的搭建与实现
数据处理与特征提取：基于开源语音数据集，完成音频数据解析，实现 MFCC 特征的提取与标准化，构建训练 / 测试数据集
模型构建与初始化：基于 Python+NumPy 从零实现全协方差 GMM 模型，通过 KMeans 算法完成模型均值、协方差、混合权重的初始化，添加空簇保护与参数正则化机制
模型训练与优化：实现 EM 算法的迭代训练逻辑，完成 E 步责任度计算、M 步模型参数更新，迭代优化模型性能
模型评估与验证：实现数值稳定的对数似然计算方法，基于测试集完成模型打分与分类，统计识别准确率并输出评估结果
